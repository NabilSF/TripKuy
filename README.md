# TripKuy

TripKuy adalah aplikasi pencarian dan pemesanan hotel yang terinspirasi dari kemudahan penggunaan Traveloka. Aplikasi ini membantu pengguna menemukan tempat menginap dengan cepat melalui tampilan yang sederhana, informasi hotel yang jelas, serta proses pemesanan yang ringkas.

Dirancang untuk penggunaan sehari-hari, TripKuy memudahkan siapa pun memilih hotel sesuai kebutuhan dengan cara yang praktis.

## Kebutuhan

1. Apache HTTP Server (XAMPP, Laragon, etc.)
2. MySQL
3. PHP 8 (atau lebih tinggi)

menggunakan framework Symfony dengan component security, twig dan Doctrine ORM
